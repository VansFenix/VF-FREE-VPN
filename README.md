VF FREE VPN — ОПИСАНИЕ САЙТА
================================

Что это?
--------
Одностраничный сайт с 6 бесплатными VLESS+Reality подписками для клиентов INCY и Happ.
Безлимитный трафик, без ограничений, без регистрации по email — только Telegram.

Возможности
-----------
✅ 6 VLESS подписок (WVFSTANDART, WVFMINI, WVFWL, WVFBYWILDTASK, WVFWILDVFROBOT, WVFBLACK)
✅ Копирование подписки в буфер обмена одним кликом
✅ Вход через Telegram (или вручную по username)
✅ Проверка подписки на канал @wildVF
✅ Игра «Орёл и Решка» с выигрышем монет
✅ Ежедневный бонус (+50 монет каждый день)
✅ Реферальная система (+50 монет за друга)
✅ Профиль с аватаркой, статистикой, балансом
✅ Загрузка своей аватарки
✅ 3D tilt на карточках подписок
✅ Плавные анимации, стеклянный дизайн (glassmorphism)
✅ Конфетти при победе в игре
✅ Админ-панель (5 кликов по лого):
   - Просмотр всех пользователей
   - Статус (активен/заблокирован)
   - Блокировка/разблокировка
   - Настройка Telegram бота

Файлы для публикации
--------------------
index.html          — главная страница
css/style.css       — стили
js/app.js           — вся логика
admin.txt           — логины и пароли админ-панели

Требования
----------
- Любой хостинг (статический)
- HTML5, CSS3, JavaScript (ES6+) — всё на клиенте
- Без сервера, без базы данных (всё в localStorage)
- Для автовхода через Telegram: нужен Cloudflare Worker + BotFather
  (но сайт работает и без этого — ручной ввод по username)
